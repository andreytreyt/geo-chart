Необходимо разработать программу на .NET c# для расчета метрик на графике XY. Программа должна уметь:

1. Предоставлять возможность загрузить данные из CSV файла. Структура файла: в одной строке файла написана одна точка - две координаты через запятую. Например:
	0.01, 10
	0.4, 20

2. Загруженные данные необходимо отобразить на графике

3. Дать возможность пользователю управлять отображением графика:
3a. Zoom / Unzoom той точки, где расположен указатель мыши
3b. Переключать оси X и Y в линейную и логарифмические шкалы
3c. Задавать min и max значениями на осях
3d. Пользователь должен видеть координаты текущего положения курсора мыши (с координатами относительно осей X, Y графика)

4. Дать возможность пользователю задавать на графике отрезок (2 точки графика), следующим образом:
4a. Пользователь нажимает левой кнопкой мыши на графике - задается первая точка
4b. Пользователь ведет мышь с зажатой левой кнопкой - рисуется стрелочка от первой точки, до текущего положения курсора
4c. В желаемой точке пользователь отпускает мышь - задается вторая точка
4d. В этом процессе точки на графике должны прилипать к графику, несмотря на то, что курсор мыши может быть уведен от графика

5. Для выбранного отрезка должны рассчитываться следующие характеристики
5a. Длина отрезка
5b. Площадь под отрезком
5c. Наклон (то есть отношение длины отрезка к высоте отрезка)
5d. Рядом нужно расположить два текстбокса в которых можно отредактировать X координаты отрезка

6. Дать возможность пользователю задавать несколько таких отрезков на графике, причем
6a. Позволять создать новые / удалить отрезки
6b. Позволять посмотреть характеристики из пункта 5 для каждого отрезка
6с. Позволить сохранить информацию об отрезках и графике в файл неважно в каком формате (сериализовать состояние редактирования)
6d. Позволить загрузить информацию об отрезках и графике из файла (десериализовать)

7. Код должен быть разбит на классы, быть разумным и аккуратным
